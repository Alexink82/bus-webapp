# Backoffice Desktop Quickstart

Короткая инструкция для администратора и диспетчера, как работать с backoffice на ноутбуке как с обычным приложением Windows.

## Что уже должно быть настроено

- Render-деплой доступен по HTTPS.
- На сервере задан `BOT_TOKEN`.
- Роли администратора и диспетчера уже назначены через `ADMIN_ID` / `ADMIN_IDS`, `DISPATCHER_ID` / `DISPATCHER_IDS` или через админ-панель.

## Первый вход

1. Откройте `admin` или `dispatcher` внутри Telegram.
2. Нажмите `Открыть в браузере`.
3. Дождитесь страницы `backoffice-login`.
4. После обмена ticket на browser-session откроется рабочая панель.

## Как установить как приложение

### Chrome / Edge

1. Откройте backoffice во внешнем браузере.
2. В меню браузера выберите `Установить приложение` / `Install app`.
3. Подтвердите установку.
4. Закрепите новое окно на панели задач Windows.

## Как это будет работать после установки

- При запуске установленного окна приложение откроется в `admin` или `dispatcher`, а не в основной брони.
- Telegram-чаты останутся в отдельном окне Telegram Desktop.
- Backoffice можно сворачивать как обычное приложение Windows.

## Если после запуска открылась главная бронь

- Убедитесь, что вы хотя бы один раз зашли во внешний `admin` или `dispatcher`.
- Проверьте, что browser-session не истекла.
- Откройте установленное приложение ещё раз после повторного входа в backoffice через Telegram.

## Если сессия истекла

- Панель вернёт вас на `backoffice-login`.
- Повторно откройте `admin` или `dispatcher` в Telegram.
- Снова нажмите `Открыть в браузере`.

## Рекомендации для операторов

- Используйте отдельное окно/PWA для `dispatcher`.
- Telegram держите открытым рядом для чатов и быстрых переходов.
- В конце смены используйте `Выйти из браузера` или `Выйти везде`.
