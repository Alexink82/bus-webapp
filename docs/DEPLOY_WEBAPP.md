# Репозиторий bus-webapp — единственный для проекта

**https://github.com/Alexink82/bus-webapp** — основной и единственный репозиторий для веб-приложения. Связи с репозиторием bus-bot нет.

## Как работать

1. **Клонируйте bus-webapp** в отдельную папку (если ещё не клонировали):
   ```bash
   git clone https://github.com/Alexink82/bus-webapp.git
   cd bus-webapp
   ```
2. Работайте в этой папке: вносите изменения, коммиты, ветки.
3. **`git push`** отправляет коммиты в **bus-webapp** на GitHub (ветка `main` или ваша текущая).
4. Деплой на Render подключается к этому репозиторию.

Никаких subtree, вторых remotes или синхронизации с bus-bot не требуется.
